# 규정 원본 파일 보관소

이 폴더는 HWP 및 DOCX 원본 파일을 보관하는 용도입니다.

## 폴더 구조

```
regulations_source/
├── new/        # 새로 받은 개정 규정 파일 (처리 대기)
├── hwp/        # HWP 원본 파일
├── docx/       # DOCX 변환 파일
└── history/    # 처리된 파일 보관
    └── 2025/
```

## 사용 방법

### 1. 개정 규정 수령
규정 담당 부서에서 받은 HWP 파일을 `new/` 폴더에 저장합니다.

**파일명 규칙**:
```
<규정코드>_제목.hwp
예: 3-1-9_교직원포상규정_20250124.hwp
```

### 2. HWP → DOCX 변환
온라인 변환기 또는 한컴오피스를 사용하여 DOCX로 변환 후 `new/` 폴더에 저장합니다.

### 3. 처리
`./scripts/batch_update.sh` 실행 시 자동으로:
- `new/` 폴더의 파일을 MD로 변환
- 해당 규정 파일 업데이트
- 처리된 파일을 `history/YYYY/` 폴더로 이동

## 주의사항

**.gitignore 설정**

`new/`, `hwp/`, `docx/` 폴더의 파일은 Git에 커밋되지 않습니다.
(용량 절약 및 불필요한 이진 파일 제외)

필요시 `.gitignore` 파일을 수정하세요.
